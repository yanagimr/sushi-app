# sushi-mulatorとは
コロナ禍で回転寿司に行けない人のための、家で寿司を注文した気になれるwebアプリです。  
UIはスシローさんの注文UIを参考にして作りました。
  

### URL
Firebaseにデプロイ中です  
URL:  <https://sushimulator.web.app/>
  
### 制作に関して
HTML,CSS,JavaScriptの勉強を始めて10日程で作りました。
